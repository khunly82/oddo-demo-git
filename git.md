# CheatSheet

## Intialisation

*Initaliser un depot*
```sh
git init
```

*Cloner un dépot*
```sh
git clone
```

## Gestion

*Configurer le depot*
```sh
git config
```

*Vérifier l'état*
```sh
git status
```

*Consulter l'historique*
```sh
git log
```

*Dire à GIT de ne plus prendre en consirération les changements du fichier*
```sh
git update-index --assume-unchanged <file>
```

## Commandes locales

*Ajouter dans le staging*
```sh
git add
```

*Retirer du staging*
```sh
git rm --cached
```

*Ajouter dans l'historique*
```sh
git commit -m "message"
```

*Ajouter dans le staging et dans l'historique*
```sh
git commit -a -m "message"  
```
